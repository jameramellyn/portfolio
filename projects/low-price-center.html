Low Price Center

A full-stack marketplace platform focused on user profiles, product listings, and personalized experiences.

Low Price Center is a full-stack marketplace application where users can browse products, create listings, and manage personalized profiles. I contributed across both the front-end and back-end to support core user features and data handling.

Full-Stack Development React / Node.js MongoDB
